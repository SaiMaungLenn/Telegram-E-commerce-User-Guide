Affiliate Program Affiliate Program

Affiliate Program ကို Admin ရော Customer ရော အားလုံးအတွက် အကျိုးရှိပါတယ်။The Affiliate Program benefits both Admins and Customers.

ဘယ်လိုအလုပ်လုပ်သလဲHow It Works

User တိုင်း ကိုယ်ပိုင် link တစ်ခုရရှိမှာဖြစ်ပါတယ်။ (/share command သုံးပြီးလဲကြည့်နိုင်ပါတယ်)။ Telegram Users အသစ်တွေက သင့်ရဲ့ link ကနေတစ်ဆင့်ဝင်ပြီး Bot ကို join လာတဲ့သူမှန်သမျှဟာ သင့်ရဲ့ users တွေအဖြစ် သတ်မှတ်မှာဖြစ်ပါတယ်။Every user gets a unique referral link (use the `/share` command to see yours). Any new Telegram user who joins the bot through your link will be counted as your referred user.

တကယ်လို့ အဲ့ဒီ users တွေထဲကတစ်ယောက်က Bot တစ်ခုရယူပြီး Upgrade လုပ်မယ်ဆိုရင် Upgrade လုပ်လိုက်တဲ့ Plan ရဲ့ 10% ကို commission fees အနေနဲ့ ရရှိမှာဖြစ်ပါတယ်။If one of your referred users gets a bot and upgrades to a paid plan, you will receive a 10% commission fee from their upgrade payment.

သင်ရဲ့ referred users lists ကိုကြည့်ဖို့ /myusers command သုံးပါ။Use the `/myusers` command to see your list of referred users.

စည်းကမ်းများRules

  • တကယ်လို့ Users က သင့်ရဲ့ link ကနေဝင်သော်လည်း၊ သင့် link ကိုမဝင်ခင်အရင်ကတည်းက Bot ကို စပြီးသားဆိုရင်တော့ အကျိုးဝင်မှာမဟုတ်ပါ။If a user clicks your link but has already started the bot before, they will not be counted as your referral.
  • တကယ်လို့ သင့် Referral Link ကနေဝင်သော်လည်း၊ တစ်ခြားတစ်စုံတစ်ယောက်ရဲ့ link ကနေဝင်ခဲ့ဖူးတဲ့ဖြစ်မယ်ဆိုရင်လဲ အကျိုးမဝင်ပါ။ ထိုသူကိုပထမဆုံးသူကသာ ပိုင်ဆိုင်ပါမည်။If a user has clicked someone else's referral link before yours, they will belong to the first referrer.
